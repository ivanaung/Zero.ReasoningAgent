# Demo Scenarios

## Scenario 1: Local Priority Briefing

Open the dashboard and ask Zero Chat for a summary of current priorities. The demo should show how Zero reasons across tasks, calendar context, reminders, and finance signals from the local workspace.

## Scenario 2: Task Creation And Follow-up

Ask Zero to create a task for a concrete date and time. Then open the Tasks or Calendar surface to show the new item in the existing workflow.

## Scenario 3: Memory-Aware Conversation

Save a memory item, such as the user's preferred timezone or normal planning style. Ask a follow-up question that depends on that saved memory to show local persistent context.

## Scenario 4: Finance Summary

Open Finance or ask Zero for a finance summary. The goal is to show practical reasoning over existing local finance data rather than a generic chat answer.

## Scenario 5: Demo Mode Walkthrough

Point out the visible Demo Mode badge, open Settings, and show that the demo is intentionally scoped to the contest-focused local reasoning workflow.
